# Anagram 2048

A small static browser game that mixes anagram solving with 2048-style sliding movement.

## Play Locally

Open `index.html` in a browser.

## Host It

This is a static site. Upload the whole `anagram-2048` folder to any static host:

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages
- Any shared hosting service that serves static files

The entry file is `index.html`.

## Project Structure

```text
anagram-2048/
  index.html
  assets/
    css/
      styles.css
    js/
      game.js
```
